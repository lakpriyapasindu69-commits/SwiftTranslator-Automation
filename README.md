# IT3040 Assignment 1 - Automation Project

This repository contains Playwright automation scripts for the Singlish to Sinhala converter (SwiftTranslator).

## Contents
- 24 Positive Functional Test Scenarios
- 10 Negative Functional Test Scenarios
- 1 UI Test Scenario (Clear Input behavior)

## How to Run
1. Install dependencies: `npm install`
2. Run tests: `npx playwright test --headed`
